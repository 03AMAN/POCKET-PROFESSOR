"Your Personalized Learning Roadmap: Track, Learn, and Achieve!"

This project creates a dynamic and interactive learning roadmap tailored to users' skills and learning goals. By selecting a skill and duration, users receive a structured roadmap with phases (Beginner, Intermediate, Advanced), each containing topics and resources. The roadmap allows users to track their progress by checking off completed topics, with progress visually represented in a real-time progress bar. It also offers the ability to export the roadmap as a markdown file for easy reference. The project integrates features like dark mode, goal setting, and collapsible sections for an enhanced user experience.
